# Stairway Calculator Pro (Canvas PWA)
Deployed via GitHub Pages. Built 2025-08-09T08:42:31.835411Z.

## Install
1. Push this folder to a public GitHub repo.
2. Enable GitHub Pages (Settings → Pages → Deploy from branch → main / root).
3. Open the URL in Safari on iPhone → Share → Add to Home Screen.

## Files
- index.html — app (fractions everywhere, PDF export, PWA registration)
- manifest.json — PWA manifest
- service-worker.js — offline cache
- icon-192.png, icon-512.png — PWA icons
